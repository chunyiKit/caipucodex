## Remote Deploy

When a remote deployment is needed for this project, use this exact command from the repository root:

```bash
DEPLOY_PASSWORD='pVS3?2+f#xJ(5nt@' ./scripts/deploy_remote.sh
```

Notes:

- This command is the project's remote deployment command.
- The password is stored here in plaintext because the project owner explicitly requested it.
- If the deployment password changes, update this file to keep the command current.
